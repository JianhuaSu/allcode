# README

## 文件夹结构

- **MSZ 文件夹（小论文相关）**
  - 包含 MSZ 方法的主要模型、基线模型以及消融实验模型。

- **MSZ_MELD 文件夹（小论文相关）**
  - 包含 MSZ 模型在 MELD 数据集下的实验结果。

- **OOD_MSZ 文件夹**
  - 包含 MSZOOD 方法的主要模型及其基线模型。

- **OOD_MSZ_ablation 文件夹**
  - 包含 MSZOOD 方法的消融实验模型。

- **TEXTOIR_ct_adb 文件夹**
  - 包含 STOID 方法的主要模型、基线模型以及消融实验模型。

- **TEXTOIR_ct_adb_mult 文件夹**
  - 包含 STOID 方法在多模态数据上的实验效果模型以及对比模型。

- **demo 文件夹**
  - 包含第六章系统的代码。

```markdown
# 项目目录结构

本项目包含多个文件夹和文件，用于组织和管理模型的代码、配置、数据和结果。以下是各个文件夹和文件的详细说明：

## 文件夹

- **backbones**
  - 存放模型的主干网络代码，通常包括预训练模型和特征提取器。

- **cache**
  - 用于存储缓存数据，以加快数据处理和模型训练的速度。

- **configs**
  - 存放模型的配置文件，包括超参数设置和训练参数。

- **data**
  - 包含用于训练和测试模型的数据集。

- **examples**
  - 提供示例代码，展示如何使用本项目中的模型和工具。

- **logs**
  - 存放训练过程中生成的日志文件，用于监控和分析模型的训练情况。

- **loss**
  - 包含定义模型损失函数的代码。

- **methods**
  - 存放模型的主要实现代码，包括模型架构和训练方法。

- **outputs**
  - 存放模型训练和测试过程中生成的输出文件，如模型权重和预测结果。

- **results**
  - 包含模型评估的结果，如准确率、召回率等性能指标。

- **tools**
  - 提供辅助工具和脚本，用于数据处理、模型评估和结果可视化。

- **utils**
  - 包含实用函数和辅助代码，供其他模块调用。

## 文件

- **requirements.txt**
  - 列出项目依赖的Python库及其版本，用于环境配置。

- **run.py**
  - 主运行脚本，用于启动模型训练、测试或其他任务。

- **examples**
  - 里面包含运行脚本。


## 使用说明

1. **环境配置**
   - 使用 `pip install -r requirements.txt` 安装项目依赖。

## 注意事项

- 确保在运行模型前，已正确配置环境并准备好数据。
- 如有疑问，请参考相关文档联系负责人。
```
